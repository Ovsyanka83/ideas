repeat as a keyword¶

Show how to do this, explain the motivation behind, copying most of the stuff from AvantPy.

repeat.py API¶

Adds repeat as a keyword to write simple loops that repeat a set number of times. That is:

repeat 3:
    a = 2
    repeat a*a:
        pass

is equivalent to:

for random_variable_name_1 in range(3):
    a = 2
    for random_variable_name_2 in range(a*a):
        pass

by:

A complete repeat statement is restricted to be on a single line
