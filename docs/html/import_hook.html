import_hook.py¶

This module contains the core functions required to create an import hook.

class ideas.import_hook.IdeasLoader(filename, module_class=None, transform_source=None, exec_=None, callback_params=None)[source]¶

A custom loader which will transform the source prior to its execution

create_module(spec)[source]¶

Return a module to initialize and into which to load.

This method should raise ImportError if anything prevents it from creating a new module. It may return None to indicate that the spec should create the new module.

exec_module(module)[source]¶

Import the source code, transform it before executing it so that it is known to Python.

class ideas.import_hook.IdeasMetaFinder(module_class=None, transform_source=None, exec_=None, callback_params=None)[source]¶

A custom finder to locate modules. The main reason for this code is to ensure that our custom loader, which does the code transformations, is used.

find_spec(fullname, path, target=None)[source]¶

finds the appropriate properties (spec) of a module, and sets its loader.

ideas.import_hook.create_hook(module_class=None, transform_source=None, exec_=None, console_dict=None, callback_params=None, name=None)[source]¶

Function to facilitate the creation of an import hook.

It sets the parameters to be used by the import hook, and also does so for the interactive console.

ideas.import_hook.remove_hook(hook)[source]¶

Function used to remove a previously import hook inserted in sys.meta_path
